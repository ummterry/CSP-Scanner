# CSP Scanner

Cash Secured Put Scanner project.

## Setup

This project uses [Poetry](https://python-poetry.org/) for dependency management.

### Prerequisites

- Python 3.9+
- Poetry

### Installation

1. Install dependencies:
   ```bash
   poetry install
   ```

### Usage

Run the main script:

```bash
poetry run python main.py
```

Or activate the shell:

```bash
poetry shell
python main.py
```
